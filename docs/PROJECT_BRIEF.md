# Project Brief

Build a polished, clickable Arabian Drilling Careers website prototype using only the provided ADC Design System handoff.

## Scope

The prototype covers Careers Home, all jobs, job detail, job family SEO pages, early careers, experienced professionals, Life at AD, EVP pages, hiring support, Talent Community, Candidate Profile, Candidate Privacy, and internal SEO inventory.

## Assumptions

- Static HTML is the right stack because the delivered prototype and design-system preview pages are static HTML/CSS.
- SuccessFactors Recruiting or the selected ATS will provide job search, listings, requisition detail, apply, saved jobs, profile, alerts, and application status.
- Detailed benefits, awards, compensation, program eligibility, and internal policies require confirmation from Talent Acquisition, HR, Legal, or Communications.
