# AD Careers Icon Pack v4 — Full Described Set

Source: Iconss(2).pptx

Total icons: 478

This version includes all extracted visual icon cells from the updated PowerPoint. Each icon has:

- web-ready transparent PNG, 512 x 512
- semantic name
- description
- alt text
- category
- use groups
- tags

The extraction merges top-level PowerPoint shapes that sit in the same visual grid cell before cropping. This helps avoid incomplete crops when an icon contains multiple parts.

Use `manifests/icons.json` as the source of truth. Select icons by `semantic_name`, `description`, `category`, `use_groups`, and `tags`. Use `manifests/by-group/*.json` for specific use cases such as oil and gas, drilling, offshore, people, recruitment, HSE, learning, and wellbeing.

Known correction: `industrial-energy-safety-014` is classified as a Saudi office worker wearing a shmagh, not a field worker.
